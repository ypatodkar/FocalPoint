Three Agents, Three Timescales
Real-time · per-turn · per-session

Prompt Agent
services/prompt_builder.py
  • Reads updated MongoDB profile before every call
  • Rebuilds system prompt from scratch each turn
  • complexity_score → vocabulary level
  • topics_to_simplify → analogy instructions
  • prospective_flags → proactive strategies
📡 Every turn · real-time
📊
Reward Agent
services/reward.py
  • Receives gaze events from previous response
  • Scalar reward — gaze replaces a human rater
  • Writes: complexity, format, topics
  • Loop closes in under one second
🔁 Session end · RSI
🧠
Meta-Agent
services/meta_agent.py
  • Triggered by POST /session/end

Six-Layer Cognitive Memory
Milliseconds to permanent — modeled on human memory
1
Sensory Memory
  • Raw WebGazer stream — ms lifetime
  • Never stored. Discarded after EMA
{ x: 432, y: 210 } → smoothed → gone
2
Working Memory
  • Live React refs — survives re-renders
  • Lost on refresh. zoneLog, sessionId
zoneLog.current = { zone_1: [t1, t2] }
3
Episodic Memory
  • MongoDB episodes — per-turn logs
  • Read by meta-agent at session end
{ turn: 3, reward: −0.5, gaze: […] }
4
Semantic Memory
  • MongoDB users — survives all sessions
  • Source of truth for Prompt Agent
{ complexity: 4, format: "bullets" }
5
Procedural Memory
  • Fixed rules in reward.py / prompt_builder.py
  • Never updated at runtime
confusion → simplify · skipped → shorten
4
Implicit RLHF — Reward Function
No labels · no clicks · reading behavior is the signal
smooth
+1.0
  • Eyes advance steadily forward
  • No backtracking detected
  • ↑ complexity_score
  • Maintain format
confusion
−0.5
  • Zone re-read 3× expected time
  • User is stuck
  • ↓ complexity · switch to bullets
  • Flag topic
skim
−0.3
  • Read time < 50% of expected
  • Scanning, not reading
  • ↓ density
  • Shorten sentences
skipped
−0.2
  • Zone never visited
  • Disengaged before reaching it
  • ↓ response length
  • Front-load key info
