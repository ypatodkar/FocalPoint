Cognitive Architecture — 6 Memory Layers
Sensory Memory
Raw WebGazer stream, milliseconds lifetime
{ x: 432, y: 210, t: 1719481234 }
Working Memory
Live session state — current turn context
zoneLog: { zone_1: [t1,t2,t3] } lastResponseId: "resp_abc"
Episodic Memory
Per-turn logs stored in MongoDB episodes collection
{ turn: 3, reward: -0.5, flag: "confusion", zone: "zone_1" }
Semantic Memory
Persistent user profile in MongoDB users collection
{ complexity: 4, format: "bullets", re_read_rate: 0.35 }
Procedural Memory
Adaptation rules baked into reward.py + prompt_builder.py
confusion → simplify skipped → shorten smooth → maintain
Prospective Memory
Future topic flags written by Antigravity meta-agent
{ topic: "recursion", action: "use_analogy" }
Implicit RLHF — Reward Function
Gaze Behaviour Flag Reward Next Response Action
Smooth read-through to end smooth +1.0 Maintain or increase depth
Re-reads same zone 3+ times in 5s confusion −0.5 Simplify: bullets, plain language
Rapid back-and-forth in small area skim −0.3 Reduce density, more whitespace
Zone never visited skipped −0.2 Shorten response length
Recursive Self-Improvement — Session End Meta-Agent

Antigravity (Google Interactions API) — runs once per session

1
Session ends → POST /session/end triggered
2
All episodes from session fetched from MongoDB (episodic memory)
3
Antigravity agent receives episodes + current user profile
4
Agent reasons deeply: pattern analysis across all turns, topics, formats
5
Returns insights + updated profile + prospective flags as JSON
6
MongoDB user profile upgraded — next session starts smarter
